# ToDope

A task management app built with a neubrutalist design aesthetic. Get things done without ceremony.

## Features

- Create, complete, and delete tasks
- Filter tasks by status: All, Active, Done
- Drag-to-reorder with priority persistence
- User authentication with email and OTP verification
- Real-time sync across sessions via Supabase subscriptions

## Tech Stack

- **React 19** with Vite
- **Supabase** for authentication, database, and real-time subscriptions
- **Bootstrap 5** for layout and utility classes
- **Framer Motion** for animations
- **IBM Plex Mono** and **Bebas Neue** for typography
